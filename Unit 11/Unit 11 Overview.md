## Unit 11: Mixed Practice I

This unit we will take a deeper dive into common graph algorithms, modifying the base BFS and DFS traversal algorithms to solve more complex problems. We will also introduce techniques for identifying problems that are not explicitly framed as graph problems but can be transformed into a graph problem and solved using similar techniques. Students will also explore additional graph algorithms including Dijkstra's and topological sort.

#### Breakout Session & HackerRank

In the problem sets, you will...

- Recognize problems that can be solved using graph algorithms
- Become familiar with common indicators that BFS, DFS, or both can be used to solve a problem
- Modify BFS and DFS algorithms to solve intermediate graph problems


### Session 1: Matrices

In this session, students will learn how to use graph concepts and algorithms to solve 2D matrix problems.

### Session 2: Advanced Graphs

In this session, students will continue to practice using BFS and DFS to solve intermediate 2D matrices problems. Students will also be introduced to additional graph algorithms including Dijkstra's Algorithm, Topological Sort, and Union Find.
