## Unit 03: Recursion, Memoization, Two-Pointer & Linear Data Structures

This unit starts our focus on arrays and strings, particularly how to manipulate them to solve common problems.

#### Lecture Session

In lecture, you will utilize recursion, memoization, along with stacks/queues/trees to structure problem-solving approaches.

In this lecture, you will...

-   Implement recursive solutions and identify appropriate base cases

-   Apply memoization to optimize recursive approaches

-   Use stacks, queues, & trees in problem-solving

#### Breakout Session & HackerRank

In the problem sets, you will...

-   Articulate the difference between array, stack, and queue data structures

-   Apply the two-pointer method to a variety of list/string problems

-   Use stacks and queues to solve problems

-   Solve complex problems involving multiple conditions, such as finding pairs with a given sum or merging intervals

### Session 1: Recursion & Memoization | Stacks, Queues, Two-Pointer

In this session we will continue to work with linear data structures like strings and arrays. Students will learn about two new linear data structures: stacks and queues. They will learn new techniques to optimally iterate through these data structures in order to solve problems including validating nesting of parentheses, reversing complex strings, finding the middle of a list, etc.

### Session 2: Stacks / Queues / Trees | Stacks, Queues, Two-Pointer

In this session we will continue to work with linear data structures like strings and arrays. Students will strengthen their ability to solve problems using stacks, queues, and the two pointer method.
